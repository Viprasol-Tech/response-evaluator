# Response Evaluator

Quality scoring and evaluation for AI responses

## Features

- Zero external dependencies (stdlib only)
- Easy-to-use CLI interface
- Professional Python implementation
- MIT licensed

## Installation

```bash
pip install -e .
```

Or clone and install:

```bash
git clone https://github.com/Viprasol-Tech/response-evaluator
cd response-evaluator
pip install -e .
```

## Usage

### Python

```python
from response_evaluator import ResponseEvaluator

result = ResponseEvaluator.process("data")
print(result)
```

### CLI

```bash
python -m response_evaluator "your input here"
```
